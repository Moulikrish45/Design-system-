Animation Principles
✓ Do
  • • Use 200-300ms eased transitions for smooth, professional feel
  • • Apply subtle opacity and position shifts for micro-interactions
  • • Maintain consistent border radius across component types
  • • Use cubic-bezier for natural, aerodynamic motion
✗ Don't
  • • No bouncy or spring animations (not enterprise-appropriate)
  • • Avoid off-screen fly-ins or dramatic entrances
  • • Don't mix different radius values in the same component
  • • Never use fully rounded (pill-shaped) buttons
